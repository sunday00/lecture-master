# 이산수학이란?

- 별개의, 개별적인, 분리된
- 연속적이 않고 분리된 값들의 연구
- 정수는 개별 숫자들의 분리체이므로 1보다 큰 가장 작은수는 무조건 2인것처럼 유한한 수를 다룸


# 집합과 논리

## 집합

### 정의: `중복되지 않는` 여러 원소들의 모임.

### 표기법
1. 나열법 : A = {1, 2, 3, 4, 5}
1. 조건 제시법 : A = { x | 1<= x < 6, x는 자연수 }
1. 벤다이어그램

### 원소의 갯수 표기 : |A| = 5

### 집합의 종류
1. 전체집합 : 논의 대상이 되는 원소 전체를 포함하는 집합. U.
2. 공집합 : 원소가 없는 집합. ø 또는 { }

### 포함관계
1. 동일관계 : A = B
1. 부분집합 : A ⊆ B : A 가 B 와 같거나 부분집합
1. 진부분집합 : A ⊂ B : A 가 B 의 부분집합일 뿐 절대 같지는 않은 부분집합
1. 원소관계 : a ∈ A : a는 A의 원소. a ∉ A : a 는 A 에 속하지 않음
1. 합집합 : A ∪ B
1. 교집합 : A ∩ B
1. 서로소 : |A ∩ B| = 0
1. 차집합 : A - B = A - ( A ∩ B )
1. 여집합 :  Ac : a ∉ A && a ∈ U 


## 명제와 논리

### 명제: 객관적 참 거짓. P

### 논리: 명제를 수식화하여 또다른 명제를 도출
1. 부정 : ~p
    - p = A가 로그인을 하였다. 
    - ~p = A가 로그인을 하지 않았다.
2. 논리곱(and) : p ^ q
    - p = A가 로그인을 하였다.
    - q = A가 권한이 존재한다.
    - p ^ q = A가 로그인을 하고 권한도 있다.
    - 둘 다 참이어야 전체가 참
3. 논리합(or) : p V q
    - p = A가 로그인을 하였다.
    - q = 이 게시판은 비회원도 작성이 가능하다.
    - p V q = A가 로그인을 했거나 이 게시판이 비회원도 글쓰기가 가능하다.
    - 둘 중에 하나라도 참이면 전체가 참
4. 조건명제(if, then) : p -> q
    - 로그인을 하였다 T, 글을 쓸 수 있다 T : 로그인을 하면 글을 쓸 수 있다 p -> q : T
    - 로그인을 하였다 T, 글을 쓸 수 있다 F : 로그인을 하면 글을 쓸 수 있다 p -> q : F
    - 로그인을 하였다 F, 글을 쓸 수 있다 T : 로그인을 하면 글을 쓸 수 있다 p -> q : T
    - 로그인을 하였다 F, 글을 쓸 수 있다 F : 로그인을 하면 글을 쓸 수 있다 p -> q : T
    - 로그인을 했다가 일단 참이면, 로그인하면 글씀가능 전체의 진리값은 글씀가능에 따라 달라지고, 
    - 로그인을 했다가 거짓이면, 로그인하면 글씀가능 전체의 진리값은 항상 참이다.
5. 역, 이, 대우
    - 역: p -> q : q -> p : 글을 쓸 수 있다 -> 로그인을 했다는 것이다.
    - 이: p -> q : ~p -> ~q : 로그인을 하지 않았다면 -> 글을 쓸 수 없다.
    - 대우 : p -> q : ~q -> ~p : 글을 쓸 수 없다면 -> 로그인을 하지 않았다는 것이다.
    - 역과 이의 진리값은 같고, 기본조건명제와 대우의 진리값도 서로 같다. (동치명제)
        - 역의 대우가 이 이기 떄문

### 명제함수와 한정자
1. 변수 x를 포함하여 진릿값을 판별할 수 있는 문장이나 수식
    - P(x) => x + 1 = 0 
    - x 가 1이면 거짓, x가 -1이면 참.
2. 전체한정자 ∀
    - ∀x P(x, x는 자연수) => x * 0 = 0
    - 논의되는 대상 모든 x에 대하여 참일때만 참인 명제.
    - 이 명제는 모든 x가 x * 0 = 0 을 만족하므로 참인 명제.
3. 존재한정자 ∃
    - ∃x P(x) => x + 1 = 0
    - 논의되는 대상 중 하나라도 참일 경우가 있으면 참.
    - 예제는 -1 인 경우에 참이 되므로 전체가 참인 명제.
