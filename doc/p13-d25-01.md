### design pattern !
- This is a solution to solve the problem in some specific context
  - 반복적인 상황에서
  - 해결해야 될 목표와 제약사항을 
  - 해결하는 일반적인 코드 방식
- 너무 구체적인 상황에서는 디자인 패턴이 아니라 그 사람이 그냥 코딩 잘 한거
- 문제를 해결하는게 아니면 그냥 컨벤션
- 누구나 반복적으로 사용할 수 있어야 하는 널리 알려진 방식이 아니면 라이브러리를 만드는게...
